Organizes M4B files via "Album Artist" property.

- J K Rowling

## Prereqs

- Deno
- ffprobe
- make

## Usage

```bash
m4b-janny *.m4b
```

## Setup

Global install (recommend)

```bash
make install
```

Executable (created in `./bin`)

```bash
make
```
